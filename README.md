# Features
----------
**Query interface:**
Keywords

Action:

- Bevaka
- Visa

Finders/Antal

- 5
- senaste
- alla

Typ

- Inbetalningar
- Utbetalningar
- Löneuttag

Where

- Datum ranges
- Summor högre än
- Innehåller

Exempel

- Bevaka utbetalningar som innehåller "mat" 
- Visa senaste utbetalningar där summan är högre än 5000
- Visa alla
- Visa utbetalningar där summan är mellan 1000 och 5000
